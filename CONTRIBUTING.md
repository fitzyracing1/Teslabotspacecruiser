# Contributing to Tesla Bot Space Cruiser

Thank you for your interest in contributing!

## Quick Start
1. Fork the repo
2. Create a feature branch
3. Make improvements (schematics, docs, simulations, 3D models)
4. Submit a Pull Request

## What We're Looking For
- Refined or new technical schematics (high-res, labeled, to-scale)
- Physics simulations or orbital models
- 3D/CAD models and renders
- Code for panel generation, optimization, or procedural design
- Hardware prototype ideas or experiments
- Documentation, animations, or educational content

## Guidelines
- Keep designs clean, professional, and well-labeled
- Version schematics consistently (e.g. TSB-01-vX.Y)
- Maintain the open, conceptual, collaborative spirit

Open an issue for questions or big ideas. Let's build something amazing together!
